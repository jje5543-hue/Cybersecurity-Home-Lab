# Cybersecurity Home Lab

## Overview
Built a virtualized cybersecurity home lab to practice 
attack and defense techniques in a safe environment.

## Tools Used
- VMware Workstation
- Kali Linux (Attack Machine)
- Windows Server 2022 (Target Machine)
- Splunk SIEM
- Nmap

## What I Built
- Configured two VMs on an isolated network
- Installed and configured Splunk SIEM for log monitoring
- Simulated network attacks using Nmap from Kali Linux
- Created automated alerts for suspicious activity
- Built a security monitoring dashboard in Splunk

## Skills Demonstrated
- Network configuration and virtualization
- SIEM deployment and management
- Threat detection and log analysis
- Attack simulation and reconnaissance
- Security monitoring and alerting
